Hi, I'm Katya Libizova! UI/UX Designer transitioning from a career in ad sales; I've gone from creating media plans to creating brands and intuitive, attractive interfaces to help users solve problems.

My work

Saavvy

Saavvy is a personal finance app that helps users save for big expenditures. This was a native mobile app design in which I created brand guidelines, wireframes, conducted user task flow analysis, and finally designed a complete set of iOS screens.

View Project

Ear Buds

Ear Buds is a music-based dating app that connects users on the basis of shared music taste. This was an extensive design project that I worked on from conception to finish, coming up with the app idea, branding, wireframes, conducting several user interviews and usability tests, and final screen designs for both iOS and Android devices along with complete and tested prototypes.

View Project

Wag World

Wag World is a location-based recommendation web app specifically catering to dog owners that gives users only dog-friendly locales by category and distance from a set location or city. For this project I took my objective which was to create a location-based app focusing on a specific user problem, from which I identified a niche and the concept for the app, conducted interviews, and took it from wireframes to a final set of screen designs.

View Project

Final Thread

Final Thread is a secondhand vintage apparel app. As my first project, the objective and core user base were all defined for me, and my job was to complete a competitive analysis, come up with branding, create key functions and unique features for the app, and design the interface from wireframing down to the final set of screens.
